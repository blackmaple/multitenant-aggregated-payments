# 《07-部署手册（K8s）.md》

## 1. 环境要求
- Kubernetes 1.28+
- Docker / Containerd
- Redis（集群或哨兵）
- PostgreSQL 14+

## 2. 部署步骤
1. 创建命名空间 `payment-prod`
2. 配置 Secret（DB、Redis、渠道密钥）
3. 部署 payment-api / tenant-api / notify-worker
4. 配置 Ingress + TLS
5. 配置 HPA（CPU 60%、最小3副本）
6. 灰度发布（Canary 10% -> 30% -> 100%）

## 3. 关键配置
- readinessProbe / livenessProbe 必须开启
- resources requests/limits 必须配置
- podDisruptionBudget 防止批量驱逐
- 生产环境禁止 latest tag
